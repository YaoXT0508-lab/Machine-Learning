### Install  
***  
scikit-learn requires:  
- Python (>= 2.6 or >= 3.3)
- NumPy (>= 1.61)
- SciPy (>= 0.9)

If you already have a working installation of numpy and scipy, the easiest way to install scikit-learn is using `pip`  
`pip install -U scikit-learn`   
#-U means update  

or `conda`:  
`conda install scikit-learn`
